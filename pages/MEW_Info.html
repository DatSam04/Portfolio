BattleShip Offline

Invidual Project

Description: Battle Ship Offline is a simple battle ship game between player and computer. When starting the game, it will generate table and ship for both side provide option to continue the game. The side that hit all ships from other side is the winner.

Responsible:
• Implement tables and rules
• Define the logic for computer, including ship placement, tracking hits and misses, as well as generating random moves
• Offer options to check the game outcome and provide information about the computer fleet

Learning:
• Demonstrate Algorithm Design for randomly placing ships without overlapping and adhere to game rule and manipulating a game grid efficiently
• Use Data Structures for a dynamic list of item
• Error Handling to prevent invalid ship placement and overlapping positions in computer movement
• Object-Oriented Programming
